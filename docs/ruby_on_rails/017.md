# How to load yml-config

1. Create a yml-file in `config/file_name.yml`
2. File structure looks like:

```yml
default: &default
  foo: <%= ENV['FOO'] %>

development:
  <<: *default

test:
  <<: *default

staging:
  <<: *default

production:
  <<: *default
```

3. In Rails console use `Rails.application.config_for(:file_name)`
