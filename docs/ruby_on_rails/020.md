# Batch with range

:use_ranges - Specifies whether to use range iteration (id >= x AND id <= y). It can make iterating over the whole or almost whole tables several times faster.

```ruby
# it will pass 1000 ids to sql-query
Person.in_batches(start: 1000).each do |relation|
  relation.update_all('age = age + 1')
end

# it will use condition id >= x AND id <= y
Person.in_batches(start: 1000, use_ranges: true).each do |relation|
  relation.update_all('age = age + 1')
end
```
